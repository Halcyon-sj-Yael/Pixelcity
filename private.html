Follow these steps to host your own Pixel City private server:

Step 1: Open Terminal/Command Prompt

Navigate to the Pixel City folder in your terminal.

Step 2: Start the Private Server

Run this command:

npm run private

Or with a custom password:

PRIVATE_PASSWORD=yourpassword npm run private
Step 3: Find Your IP Address

When the server starts, it will display your IP address in the terminal. Look for:
